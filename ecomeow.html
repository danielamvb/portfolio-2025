ECO-MEOW

Game Group Project

Eco-Meow is a 2D educational game that encourages children to recycle while having fun. Built using the Canvas API and CSS animations, with some visual assets sourced from online repositories, the game allows players to select a cat character and help clean up litter by sorting it into the correct recycling bins. As a reward for correctly recycling items, players earn seeds, which can be planted to grow trees, reinforcing the importance of sustainability. Inspired by the Eco-Escolas initiative, Eco-Meow transforms environmental awareness into an engaging and interactive experience.
